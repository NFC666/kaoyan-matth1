# 求导路径选择策略

## 决策流程

拿到一个求导任务时，按以下决策树选择正确的求导方法：

```mermaid
flowchart TD
    A[需要求导] --> B{函数形式?}
    B -->|显函数| C[基本公式 + 四则/链式法则]
    B -->|分段函数分段点| D[必须用导数定义！]
    B -->|隐函数 F⁡x,y=0| E[两边对 x 求导<br>y 的项乘 y']
    B -->|幂指函数 f⁡x^g⁡x| F[对数求导法<br>或化为 e^{g·ln f}]
    B -->|参数方程| G[dy/dx = dy/dt / dx/dt<br>二阶须再除以 dx/dt]
    B -->|变限积分| H{被积函数含 x?}
    H -->|不含| I[Leibniz 公式：<br>f⁡b⁡x·b'⁡x - f⁡a⁡x·a'⁡x]
    H -->|含 x| J[先提出 x 或变量代换]
    J --> I
```

## 各类方法的判断信号

| 函数形式 | 识别信号 | 首选方法 |
|---------|---------|---------|
| 分段函数 | 分段点处求导 | **导数定义**（唯一正确方法） |
| $F(x,y)=0$ | 方程含 $x,y$ 混合 | 两边对 $x$ 求导 |
| $f(x)^{g(x)}$ | 底和指数都含变量 | 对数求导法 |
| $\begin{cases}x=\varphi(t)\\y=\psi(t)\end{cases}$ | 参数给出 | $\frac{dy/dt}{dx/dt}$ |
| $\int_{a(x)}^{b(x)}$ | 积分号上下限含变量 | Leibniz 公式 |
| 乘积 $u(x)v(x)$（高阶） | 求 n 阶导 | 莱布尼茨公式 |

## 关键原则

1. **分段点求导**：唯一正确的是定义法，不能用公式代
2. **幂指函数**：先取对数，没有第二条路
3. **隐函数二阶导**：先代入 $(x_0, y_0, y'_0)$ 简化一阶导等式，再求导
4. **变限积分含 $x$**：$x$ 对积分变量 $t$ 是常数，必须提出来
5. **参数方程二阶导**：必须再除以 $\varphi'(t)$，不是直接 $\psi''/\varphi''$
