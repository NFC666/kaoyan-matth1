# 题型七：渐近线求法

## 识别特征

- 题干问"曲线 $y=f(x)$ 的渐近线"
- "下列哪条是渐近线"的选择题

## 解题流程

```mermaid
flowchart TD
    A[识别：求渐近线] --> B{检查方向}
    B --> C[水平渐近线]
    B --> D[垂直渐近线]
    B --> E[斜渐近线]
    C --> C1[计算 lim_{x→±∞} f⁡x]
    C1 --> C2{极限 = 常数 C?}
    C2 -->|是| C3[y = C 为水平渐近线]
    C2 -->|否| C4[无水平渐近线]
    D --> D1[找间断点 x₀]
    D1 --> D2[计算 lim_{x→x₀} f⁡x]
    D2 --> D3{极限 = ∞?}
    D3 -->|是| D4[x = x₀ 为垂直渐近线]
    E --> E1[计算 k = lim_{x→∞} f⁡x/x]
    E1 --> E2{k 存在且 ≠ 0,∞?}
    E2 -->|是| E3[计算 b = lim f⁡x - kx]
    E2 -->|否| E4[无斜渐近线]
    E3 --> E5{b 存在?}
    E5 -->|是| E6[y = kx+b 为斜渐近线]
```

## 通法步骤

| 类型 | 方法 |
|------|------|
| 水平渐近线 | $\lim\limits_{x \to \pm\infty} f(x) = C$ → $y = C$ |
| 垂直渐近线 | 找间断点 $x_0$，$\lim\limits_{x \to x_0} f(x) = \infty$ → $x = x_0$ |
| 斜渐近线 | $k = \lim\limits_{x \to \infty} \frac{f(x)}{x}$, $b = \lim\limits_{x \to \infty} [f(x) - kx]$ |

## 常见陷阱

- 忘记分别考虑 $x \to +\infty$ 和 $x \to -\infty$——两边可能有不同的水平/斜渐近线
- 水平渐近线和斜渐近线在同侧不能共存（水平存在则斜不存在）
- 垂直渐近线：间断点需要验证 $\lim f(x) = \infty$，不是所有间断点都是垂直渐近线

## 经典母题

> **题目**（真题）：求曲线 $y = x + \frac{\ln x}{x}$ 的渐近线。

**解析**：
- **水平**：$\lim\limits_{x \to +\infty} y = +\infty$，无水平渐近线
- **垂直**：$x \to 0^+$ 时 $\frac{\ln x}{x} \to -\infty$，故 $x=0$ 是垂直渐近线
- **斜**：$k = \lim\limits_{x \to +\infty} \frac{x + \frac{\ln x}{x}}{x} = 1$, $b = \lim\limits_{x \to +\infty} \left(x + \frac{\ln x}{x} - x\right) = 0$

故斜渐近线为 $y = x$。
